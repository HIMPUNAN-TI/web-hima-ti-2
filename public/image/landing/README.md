# Landing Page Assets

This folder contains images and assets for the Himaprodi TI landing page.

## Required Images

Please add the following images to make the landing page complete:

1. **logo.png** - Himaprodi TI logo (40x40px recommended)
2. **hero-bg.jpg** - Background image for the hero section (1920x1080px recommended)
3. **about-1.jpg** - First about section image (300x200px recommended)
4. **about-2.jpg** - Second about section image (300x200px recommended)
5. **about-3.jpg** - Third about section image (300x200px recommended)

## Event Images

Event images are loaded from `public/image/events/posters/` directory. The website will automatically use the poster images from the events in your database.

## Fallback Images

The website includes fallback SVG placeholders for missing images, so it will display properly even without the actual image files.

## Image Specifications

-   **Logo**: 40x40px, PNG format with transparent background
-   **Hero Background**: 1920x1080px, JPG format, optimized for web
-   **About Images**: 300x200px, JPG format, optimized for web
-   **Event Posters**: 400x300px, JPG/PNG format, optimized for web

All images should be optimized for web use (compressed, appropriate format) to ensure fast loading times.
